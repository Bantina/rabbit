#Rabbit Project

A scaffold based on [webpack](https://webpack.github.io/) and [Vue.js](https://vuejs.org.cn/).

###Run the app

```bash
npm install
npm start
```
 
###Major module dependencies
* [htttp-hash-router](https://github.com/Matt-Esch/http-hash-router)
* [st](https://github.com/isaacs/st)
